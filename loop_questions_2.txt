
Basic Python Loop Questions
===========================

For Loop Questions
------------------
1. Write a `for` loop to print numbers from 5 to 15.
2. Use a `for` loop to print each item in a list of colors (`["red", "green", "blue"]`).
3. Write a `for` loop to calculate the total of a list of numbers (`[5, 10, 15, 20]`).
4. Create a `for` loop to print the first 5 multiples of 3.
5. Use a `for` loop to print each word in the sentence `"Python is fun"` on a new line.
6. Write a `for` loop to add all the even numbers in a list (`[1, 2, 3, 4, 5, 6]`).
7. Create a `for` loop to print all numbers from 1 to 10, skipping every other number.
8. Use a `for` loop to count the number of items in a list (`["apple", "banana", "cherry"]`).
9. Write a `for` loop to print the square of each number in a range from 1 to 5.
10. Use a `for` loop to repeat the word `"Hello"` 5 times.

While Loop Questions
--------------------
1. Write a `while` loop to print numbers from 1 to 5.
2. Use a `while` loop to print a countdown from 5 to 1.
3. Write a `while` loop to print `"Learning Python"` 3 times.
4. Create a `while` loop to add numbers from 1 to 10.
5. Write a `while` loop to print each character in the string `"Loops"`.
6. Use a `while` loop to find the sum of all numbers in a list (`[2, 4, 6, 8]`).
7. Write a `while` loop to print all numbers between 1 and 10 that are divisible by 2.
8. Create a `while` loop to keep asking the user for input until they type `"stop"`.
9. Write a `while` loop to find the product of all numbers from 1 to 5.
10. Use a `while` loop to print numbers from 1 to 10, but stop early if the number is greater than 7.
